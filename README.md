# ember-cli-rx

This addon provides helpers to integrate RxJS with Ember. More details soon.

## Installation

From within your Ember CLI application (must be > 0.0.42), run the
following:

    npm install --save-dev ember-cli-rx
    ember generate ember-cli-rx
